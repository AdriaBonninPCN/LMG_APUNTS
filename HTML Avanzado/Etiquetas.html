Etiquetas HTML avanzado

Etiqueta <center>

La etiqueta <center> se utilizaba em versiones antiguas de HTML para centrar contenido dentro de una página web, tanto texto como imágenes o otros elementos.

Está obsoleta en HTML5 y se tendria que evitar su uso. En lugar de <center>, hoy en día se usa CSS para centrar elementos de manera más eficiente.

Ejemplo:

    <center>
     <h1>Bienvenidos a mi sitio web>/h1>
    </center>

    IMPORTANTE: No usar más esta etiqueta, ya que está obsoleta.

Etiqueta <div>

  • La etiqueta <div> (abreviatura de "division") se utiliza para crear contenedores de bloque en una página HTML. Es una etiqueta de nivel de bloque, la cual cosa significa que ocupa todo el ancho disponible i comienza en una línea nueva.
  • No tiene significado semántico en si misma; simplemente agrupa cotros elementos, como texto, imágenes o otras etiquetas y suele ser estilizada con CSS.
  • Es útil para organizar i diseñar diferentes secciones de una página web.
    <div>
    <h2>Sección 1</h2>
    <p>Esto es un párrafo en un div :).</p>
    </div>

Buenas prácticas para usar divs en HTML:
  • Uno de los errores más comunes es el uso excesivo de <div> para cualquier cosa, cuando existen etiquetas semánticas como <header>, <section>, <article>, <footer>, ....
  • Utilizar <div> para agrupar elementos que forman una unidad lógica o visual dentro de una página, pero siempre que no haya una etiqueta semántica más adecuada.

Etiqueta <span>

  • La etiqueta <span> es una etiqueta inline (en línea) que sirve para estilizar o manipular pequeñas porciones de texto o contenido dentro de una página. A diferencia de <div>, no ocupa todo el ancho disponible, si no que ocupa sólo el espacio necesario.
  • También es una etiqueta no semántica, usada principalmente para aplicar estilos o funcionalidades a un grupo específico de elementos dentro de un texto.

    • Ejemplo:

      <p>Esto es un
       <span style="color: red";>texto en rojo</span>.
      </p>

    Esto es un texto en rojo.
