Validación de Formularios

HTML5 introduce una serie de nuevos atributos i tipos de entrada para validar los formularios de manera nativa, sin necesidad de JavaScript.

Artibuto "required":

  • Marca un campo como obligatorio.

    • Ejemplo:

      <input type="text" name="nombre" required placeholder ="Escribe tu nombre">

Artibuto "pattern":

  • Permite definir un patrón con expresiones regulares para validar la entrada del usuario.

    • Ejemplo:

      <input type="text" name="teléfono" pattern="\d{9}" placeholder="Escribe tu teléfono">
