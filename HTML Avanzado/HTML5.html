HTML5 i su Evolución

Es una evolución importante respecto a las versiones anteriores de HTML, introduciendo nuevas etiquetas i funcionalidades que permiten una mejor gestión de contenido multimedia, formularios avanzado y mejoras en la semántica de la página web.

Etiquetas en HTML5:

  1. <section>: Se usa para agrupar contenido relacionado dentro de una página.
  2. <article>: Representa un bloque de contenido autonomo, como un articulo de un blog, una notícia o un post que puede existir independientemente del sitio web.
  3. <aside>: Define contenido relacionado con el contenido principal pero no esencial. Normalmente se utiliza para las barras laterales, anuncios o información complementária.
  4. <header> i <footer>: Etiquetas para definir cabezeras i pies de página.
  5. <nav>: Etiqueta específica para las barras de navegación del sitio web.
  6. <figure> i <figcaption>: Agrupa elementos multimedia (como imágenes o vídeos) i sus descripciones.
Buenas prácticas para utilizar elementos de estructura:
  • Utilizar elementos semánticos para organizar mejor la página. Esto hace que sea más comprensible para los navegadores, buscadores i lectores de pantalla.
  • Evitar el uso excesivo de "divs" i aprovechar las etiquetas <section>, <article> i <aside> para hacer que la página sea más semántica.
  • Proporcionar enlaces de navegación claros dentro de los elementos <header> i <footer> para mejorar la experiencia del usuario i la accesibilidad.
  • No abusar de <aside>, ya que este elemento tiene que contener información complementaria i no esencial para el contenido principal.
