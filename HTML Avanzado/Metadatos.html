Metadatos en HTML

Los metadatos son información que se coloca dentro del documento HTML para describir características del documento como el autor, la descripción, las palabras clave, las intrucciones para motores de busqueda i navegadores.

Los metadatos se inluyen dentro de la etiqueta <head> del documento HTML, pero no se muestran directamente en la página web. Estos datos ayudan a los navegadores, motores de busqueda i aplicaciones a comprender mejor el contenido de la página.

Charset (Conjunto de carácteres)

  • Define el conjunto de carácteres utilizando el documento, como UTF-8, que soporta casi todos los carácteres

Ejemplo:

<meta charset="UTF-8">

Viewport (Responsive Design)

  • Ayuda a configurar la visualización de una página web en dispositivos móviles, garantizando que el contenido se adapte correctamente a diferentes medidas de pantalla.

    • Ejemplo:

    <meta name="viewport" content="width=device-width, initial-scale=1.0"

Descripción de la página

  • Proporciona una breve descripción del contenido de la página. Esta información se utiliza para motores de búsqueda para mostrar descripciones en los resultados de busqueda.

Ejemplo:

<meta name="descrption" content="Este sitio web ofrece información sobre HTML5"

Palabras clave (Keywords)

  • Define un conjunto de palabras clave que describen el contenido de la página. Incluso si no se usa tanto por los motores de busqueda actualmente, puede ser útil para otras aplicaciones.

Ejemplo:

<meta name="keywords" content="HTML5. desarrollo web, CSS, JavaScript">

Autor de la página

  • Indica el nombre del autor o empresa responsable de página web.

Ejemplo:

<meta name="author" contento="Adrià Bonnin">

Intrucciones para Robots de Busqueda

  • Indica a los motores de busqueda como indexar i rastrear el contenido de la página.

<meta name="robots" content="index, follow">

Open Graph Tags (per a redes sociales)

  • Los Open Graph Tag son metadatos utilizados para las redes sociales (como Facebook o Twitter) para mostrar previsualizaciones personalizadas cuando se comparte un enlace.

<meta property="og:title" content="Título de la página">
<meta property="og:description" content="Descripción de la página">
<meta property="og:image" content="imagen.jpg">
<meta property="og:url" content="https://www.exemple.com">

Metadatos para aplicaciones móviles

  • Puedes utilizando metadatos para mejorar como se ve la página cuando se guarda en la pantalla de inicio en el dispositivo móvil.

<link rel="appla-touch-icon" href="icon.png"

Favicon

  • Incluso si no es un <meta>, el favicon es un metadato visual imporatnte que se muestra en la pestaña del navegador.

<link rel="icon" href="favicon.ico" type="image/x-icon">

lang(atributo)

  • El atributo lang se define dentro de la etiqueta <html> para especificar el idioma general de la página, o dentro de cualquier otra etiqueta HTML para marcar un idioma específico dentro de una parte del contenido.

<!DOCTYPE html>
<html lang="ca">
...
</html>

Buenas prácticas para hacer formularios en HTML:
  • Utiliza el conjunto de carácteres UTF-8 para garantizar que todos los carácteres especiales se muestren correctamente en todos los dipositivos i navegadores.
  • Incluye una descripción relevante de la página para ayudar los motores de buscar a comprender mejor el contenido y mejorar tu clasificación.
  • Optimiza pera móviles utilizando la etiqueta viewport para asegurar que la página se adapte correctamente a diferentes dispositivos.
  • Incluye Open Graph Tags si quieres que tu página se muestre de manera atractiva cuando se coparta en redes sociales.
  • Utiliza metadatos de robots para controlar como quieres que los motores de busqueda indexen o no indexen tu página.
