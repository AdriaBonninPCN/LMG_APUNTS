Formularios con HTML

Los formularios son una de las herramientas más importantes de HTML porque permiten a los usuarios enviar datos a un servidor. Con HTML5, los formularios han ganado nuevas funcionalidades que mejoran su usabilidad i validación.

Un formulario se crea con la etiqueta <form>, que actua como el contenedor de todos los elementos de entrada (texto, botones, selecciones, ...).

Cada campo del formulario se envia como datos a un servidor cuando el usuario lo envia.

Ejemplo:

    <form action="submit.php" method="POST">
     <!-- Campos del formulario -->
    </form>

<input>

La etiqueta <input> es el elemento más versátil para crear campos de entrada. El tipo de campo se define con el atributo "type"

Ejemplo:

    <input type="text" name="nombre" placeholder="Escribe tu nombre">

    <input type="email" name="email" placeholder="Escribe tu correo">

    <input type="password" name="password" placeholder="Escribe tu contraseña">

<label>

La etiqueta <label> se utiliza para etiquetar campos de formulario i mejorar la accesibilidad.

Ejemplo:

    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre">

<textarea>

Para textos largos, como comentarios o descripciones, se utiliza <textarea>, que permite un area de texto más grande.

Ejemplo:

    <textarea name="comentario" rows="4" cols="50" placeholder="Escribe tu comentario aquí"></textarea>

<select> y <option>

<select> se usa para crear listas desplegables de selección. Cada opción dentre de la lista se define com <option>.

Ejemplo:

    <label for="ciudad">Selecciona tu ciudad:</label>
    <select id="ciudad" name="ciudad">
     <option value="barcelona">Barcelona</option>
     <option value="madrid">Madrid</option>
     <option value="valencia">Valencia</option>
    </select>

<multiple>

El atriubto <multiple> se usa para listas desplegables de selección. Permite seleccionar múltiples opciones en una lista de selección.

Ejemplo:

    <select name="intereses" multiple>
    <option value="programacion">Programación</option>
    <option value="diseno">Diseño</option>
    <option value="marketing">Marketing</option>
    </select>

<button>

<button> se utiliza para crear botones de enviamiento o otras acciones dentro del formulario.

Ejemplo:

    <button type="submit">Enviar</button>

Radio buttons

Los Radio buttons permiten seleccionar una opción entre un cojunto de posibilidades.

Ejemplo:

    <label><input type="radio" name="sexo" value="hombre"> Hombre</label>
    <label><input type="radio" name="sexo" value="mujer"> Mujer</label>

<checkboxes>

Las checkboxes permiten seleccionar diversas opciones independientes.

Ejemplo:

    <label><input type="checkbox" name="subscripcion" value="noticias"> Suscribeme a las noticias</label>
    <label><input type="checkbox" name="subscripcion" value="ofertas"> Subscribeme a las ofertas</label>

Buenas prácticas crear formularios en HTML. Siguiendo estas recomendaciones mejoraremos la experiencia del usuario, la accesibilidad, la seguridad i la eficiencia de los formularios para nuestras páginas web.

Buenas prácticas para hacer formularios en HTML:
  • Usar la etiqueta <label> para mejorar la accesibilidad.
  • Utilizar los tipos de entrada adecuados.
  • Definir el atributo required en los campos obligatorios.
  • Añadir la validación nativa con pattern i min/max.
  • Usar placeholder para proporcionar pistas al usuario.
  • Organizar los campos del formulario en grupos lógicos.
  • Proporcionar información clara sobre los errores.
  • Utilizar autocomplete para mejorar la experiencia del usuario.
