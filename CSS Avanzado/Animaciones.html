Animaciones y Transiciones

Las animaciones y transiciones en CSS son herramientas poderosas que permiten añadir movimiento e interacción visual a las páginas web, mejorando la experiencia del usuario. Con estas técnicas, puedes controlar cómo los elementos cambian sus propiedades (como color, posición u opacidad) de manera suave y fluida.

  1. Uso de transiciones (transition): Las transiciones CSS permiten que los cambios en las propiedades de un elemento, como el color, la opacidad, el tamaño o la posición, sean suaves en lugar de bruscos. Cuando una propiedad cambia, puedes definir la velocidad del cambio utilizando transition.

    2. Sintaxis básica de transición

    elemento {
     transition: propietat durada funció-temporització retard;
    }

    1. Propiedad: La propiedad CSS que quieres animar (ej. background-color, transform, opacity, etc.).
    2. Duración: Cuánto tiempo durará la transición (ej. 0.5s, 2s).
    3. Función de temporización: Controla cómo será la velocidad de la transición (ej. ease, linear, ease-in-out).
    4. Retraso: Opcional, define cuánto tiempo tardará en comenzar la transición (ej. 1s).

    Ejemplo de transición simple

    button {
     background-color: lightblue;
     transition: background-color 0.3s ease;
    }
    button:hover {
     background-color: darkblue;
    }

  2. Propiedades de transición (transition-property): Si quieres animar varias propiedades simultáneamente, puedes especificarlas individualmente con transition-property o utilizar all para aplicar la transición a todas las propiedades que cambien.

    3. Ejemplo animando varias propiedades

    div {
     width: 100px;
     height: 100px;
     background-color: lightgreen;
     transition: width 1s ease, background-color 0.5s ease-in-out;
    }
    div:hover {
     width: 200px;
     background-color: darkgreen;
    }

  3. Animaciones clave con @keyframes: Las animaciones en CSS permiten crear movimientos más complejos que las transiciones. Utilizan la regla @keyframes para definir puntos intermedios del movimiento o cambio de propiedad, y se aplican a través de la propiedad animation.

    4. Sintáxis de @keyframes

    @keyframes nom-animacio {
     0% { propietat: valor-inicial; }
     100% { propietat: valor-final; }
    }

    Sintáxis de animation


    element {
     animation: nom-animacio durada funció-temporització retard cicles
     direcció;
    }

    • Nombre de la animación: El nombre de la animación definida con @keyframes.
    • Duración: El tiempo que durará la animación.
    • Función de temporización: Controla la velocidad de la animación (linear, ease, etc.).
    • Retraso: Opcional, cuánto tiempo tardará en comenzar la animación.
    • Ciclos: Cuántas veces se repetirá la animación (infinite para repetirla indefinidamente).
    • Dirección: Opcional, si la animación se reproduce de manera normal o inversa (normal, reverse, alternate).
  4. Propiedades de animación (animation-duration, animation-delay): Las propiedades de animación te permiten personalizar cómo se comporta una animación. Las más importantes son:
    • animation-duration: Controla cuánto tiempo dura la animación.
    • animation-delay: Define un retraso antes de que la animación comience.
    • animation-iteration-count: Controla cuántas veces se repetirá la animación (1, 3, infinite).
    • animation-direction: Define si la animación se reproduce hacia adelante (normal), hacia atrás (reverse) o alterna (alternate).

    5. .cuadro {
     width: 100px;
     height: 100px;
     background-color: #3498db;
     position: relative;
     animation-name: moure;
     animation-duration: 2s; /* Durada de la animación */
     animation-delay: 1s; /* Retardo antes de empezas */
     animation-iteration-count: 3; /* Repetición de la animación */
     animation-direction: alternate; /* Dirección alternada */
    }
