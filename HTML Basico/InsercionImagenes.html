Inserción de Imágenes

  • La etiqueta <img> se utiliza para insertar imágenes en una página web.
  • No tiene etiqueta de cierre, ya que es un elemento autocerrado.
  • Es esencial proporcionar información sobre la imágen para que se muestre correctamente i mejorar la accesibilidad.

Atributos de imágenes:

  1. src (source)
    • Define la fuente o el camino de la imágen.
    • Puede ser un camino relativo o una URL completa.
  2. alt (alternate text)
    • Proporciona un texto alternativo que se muestra si la imágen no se carga o es leída por lectores de pantalla.
    • Es fundamental para la accesibilidad i SEO
  3. width i height:
    • Especifica el ancho (width) i el largo de la imágen en píxeles o porcentajes.
    • Es importante para controlar la medida de la imágen i la página.
Buenas prácticas para imágenes:
  • Siempre utilizar el atributo alt para garantizar una buena accesibilidad y seo.
  • Evitar insertar imágenes muy grandes sin reducir su tamaño, ya que esto puede afectar al rendimiento de la página.
  • Si las imágenes son responsivas (se adaptan a diferentes dispositivos), utiliza porcentajes en lugar de píxeles para width i height

Ejemplo:

<img src="https://pic2-cdn.creality.com/crealityCloud/common/4a17ef41ef476f9ae961be878abb2dc1.webp" alt="Hatsune Miku" widht="300" height="300">
