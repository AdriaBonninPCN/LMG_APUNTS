Tablas Avanzadas en HTML

Etiqueta <thead> (table head):

  • Define el encabezamiento de la tabla, agrupando las filas del encabezamiento (habitualmente las que contienen <th>).

Etiqueta <tbody> (table body):

  • Agrupa el cuerpo de la tabla, es decir, las filas de datos. Se suele usar para separar datos de los encabezamientos.

Etiqueta <th> (table header):

  • Se utiliza para crear la celda de encabezamiento en una tabla. El texto dentro de <th> aparece en negrita i centrado por defecto.

Ejemplo:

<table border="1" cellpadding="10" cellspacing="5">
 <thead>
  <tr>
   <th>Producte</th>
   <th>Preu</th>
  </tr>
 </thead>
 <tbody>
 <tr>
  <td>Poma</td>
  <td>1.50€</td>
 </tr>
 <tr>
  <td>Plàtan</td>
   <td>1.00€</td>
 </tr>
 </tbody>
</table>

La tabla de arriba sería representada así:

Producto Precio
Manzana 1.50€
Plátano 1.00€

Buenas prácticas para tablas:
  • Utilizar <thead> i <tbody> para organizar las tablas, especialmente en tablas más grandes, con tal de separar claramente el encabezamiento del cuerpo de la tabla
  • Mantener las tablas simples para datos tabuladas i utilizar CSS para el diseño más avanzado de las tablas (colores, medidas de la fuente, ...).
