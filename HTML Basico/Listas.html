Listas HMLT

Listas No Ordenadas (<ul>):

  1. Se utilizan para listas senzillas de elementos sin orden específico.
  2. Los elementos se muestran en viñetas o otros símbolos antes de cada ítem.
  3. Cada elemento de la lista se define con la etiqueta <li> (list item).

Ejemplo:

<ul>
<li>Primer ítem</li>
<li>Segon ítem</li>
<li>Tercer ítem</li>
</ul>

La lista de arriba sería representada así:

  • Primer ítem
  • Segon ítem
  • Tercer ítem

Listas Ordenadas (<ol>):

  1. Se utilizan para listas en las que el orden de los elementos el relevante.
  2. Los elementos se enumeran automáticamente (1, 2, 3, ...).
  3. Cada elemento de la lista también se define con <li>.
<ol>
<li>Primer ítem</li>
<li>Segon ítem</li>
<li>Tercer ítem</li>
</ol>

La lista de arriba sería representada así:

  1. Primer ítem
  2. Segon ítem
  3. Tercer ítem

Atributos de las Listas

  1. Atributo "type" (para listas ordenadas <ol>):
    • Se puede utilizar para definir el tipo de numeración de la lista:
      • 1: Números (predeterminado).
      • A: Mayúsculas.
      • a: Minúsculas.
      • I: Números romanos en mayúsculas.
      • i: Números romanos en minúsculas.

  2. Atributo "start" (para listas ordenadas <ol>):
    • Define el número o la letra para comenzar la numeración.

Ejemplo:

<ol type="A">
 <li>Primer ítem</li>
 <li>Segon ítem</li>
</ol>

<ol start="5">
 <li>Ítem número 5</li>
 <li>Ítem número 6</li>
</ol>

La lista de arriba sería representada así:

  1. Primer ítem
  2. Segon ítem
  1. Ítem número 5
  2. Ítem número 6
