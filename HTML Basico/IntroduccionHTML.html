Introducción a HTML

¿Qué es HTML?

  1. HTML (HyperText Markup Language) es el lenguaje de marcas para crear páginas web.
  2. Define la estructura de la página con etiquetas que el navegador interpreta para mostrar contenido.

Breve historia

  1. Creado por Tim Berners-Lee en el 1999
  2. HTML 2.0 (1995): Primer estandar oficial
  3. HTML 5 (2014): Última versión con nuevas funcionalidades como el soporte para vídeos, audio i gráficos.
