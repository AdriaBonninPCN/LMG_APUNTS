Etiquetas Básicas

<title>

Define el titulo para que aparezca en la pestaña del navegador o en los resultados de los buscadores.

Ejemplo:

<head>
 <title>Etiquetas Básicas</title>
</head>

Este ejemplo define el título de la página a: "Etiquetas Básicas", como esta misma página.

<h1> a <h6>

Utilizados para definir títulos i subtítulos en la página web.

Ejemplo:

<body>
 <h1>Título 1</h1>
 <h2>Título 2</h2>
 <h3>Título 3</h3>
 <h4>Título 4</h4>
 <h5>Título 5</h5>
 <h6>Título 6</h6>
</body>

<p>

Se utiliza para definir bloques de texto o parágrafos.

Ejemplo:

<body>
 <p>Bloque de texto<p>
</body>

<br>

Fuerza el texto a empezar en una nueva línea.

Ejemplo:

<body>
 <p>Linea 1<br>Linea 2</p>
</body>

El texto de arriba sería representado así:

Linea 1
Linea 2

& nbsp;

Inserta un espacio en blanco que no se rompe entre líneas.

<body>
 <p>Proyecto& nbsp;1</p>
</body>

El texto de arriba sería representado así:

Proyecto 1
