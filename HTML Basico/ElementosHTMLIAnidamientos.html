Elementos HTML i Anidamientos

Diferencia entre Etiquetas i Elementos

Etiqueta (tag):

  • Es la parte que define el inicio y el final de un componente HTML
  • Se dilimita con símbolos <>.
    Ejemplos: <p>, </p>, <h1>, </h1>.

Ejemplo:

<p>

Elemento:

  • Un elemento incluye la etiqueta de abertura, el contenido (si hay), i la etiqueta de cierre.
  • Los elementos pueden estar vacíos o tener otro dentro.

Ejemplo:

<p>Esto es párrafo</p>

Anidamientos

Elementos anidados

  • Anidamiento se refiere a colocar un elemento HTML dentro de otro elemento.
  • Es importante seguir correctamente el orden de las etiquetas: la primera que se abre es la última que se cierra.
<!DOCTYPE html>
<html>
 <head></head>
 <body></body>
</html>
