Estructura Básica de un documento HTML

Estructura General

  1. Un documento HTML se basa en una estructura jerárquica de etiquetes que definen las diferentes partes de una página.
  2. Cada página HTML sigue una estructura similar, empezando por <!DOCTYPE>, seguido de las etiquetas <html>, <head>, <body>.

Ejemplo:

<!doctype html>
<html>
 <head></head>
 <body></body>
</html>
