Enlaces en HTML

  • La etiqueta <a> se utiliza para crear enlaces (links) a otra páginas web, documentos, secciones dentro de la misma página o cualquier otro recurso.
  • El elemento esencial para los enlaces es el atributo href que define la URL a la cual redirige al usuario.
  • El texto entre la etiqueta de abertura i la de cierre es el que los usuarios veran i podran clicar para ser redirigidos.

Atributos de Enlaces:

  1. href (hypertext reference):
    • Define la URL de destinación del enlace.
    • Puede ser un enlace absoluto (una URL completa) o relativo (camino dentro de un mismo dominio).
  2. target:
    • Define donde se abrirá el enlace: en la misma pestaña o en un nuevo marco.
    • Los valores más habituales són:
      • _blank: Abre el enlace en una pestaña nueva.
      • _self: Abre el enlace en la misma pestaña (valor predeterminado)
Buenas prácticas para enlaces:
  • Utiliza enlaces descriptivos en lugar de genéricos como: "Clica aquí". Esto mejora la accesibilidad i el SEO.
    • Correcto: "Visita página de Contacto"
    • Incorrecto: "Clica aquí"
  • Utiliza target="_blank" sólo cuando sea necesario, ya que abrir enlaces en nuevas pestañas puede confundir a algunos usuarios.
  • Siempre asegurate de que los enlaces sean accesibles añadiendo una descripción adecuada para los usuarios con lectores de pantalla.

Ejemplo:

<a href="https://www.google.com">Buscar en google</a>
<a href="#capitulo1">Ir al capítulo 1</a>
<h2 id="capitul1o">Capítulo 1</h2>
<a href="https://www.ejemplo.com" target="_blank">Abre en una nueva pestaña</a>

El ejemplo de arriba sería representado así:

Buscar en google
Abre google en una nueva pestaña
