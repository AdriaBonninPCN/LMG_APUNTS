Texto Formateado

<b> i <strong>: Negrita

<b> aplica negrita sin indicar ninguna importáncia semántica.
<strong> aplica negrita y indica importancia semántica.

Ejemplo:

<b>Hola mundo</b>

El texto de arriba sería representado así:

Hola mundo

<i> i <em>: Cursiva

<i> aplica cursiva sin implicar enfasis.
<em> aplica cursiva con enfasis, implicando que el texto tiene importáncia.

Ejemplo:

<i>Hola mundo</i>

El texto de arriba sería representado así:

Hola mundo

<u>: Subrallado

Se utiliza para subrallar texto.

Ejemplo:

<u>Hola mundo</u>

El texto de arriba sería representado así:

Hola mundo

<strike> o <s>: Text ratllat

Aplica un rallado al texto, indicado que está borrado o no es relevante.

Ejemplo:

<s>Hola mundo</s>

El texto de arriba sería representado así:

Hola mundo

<sup>: Superíndex

Se utiliza para elevar texto como en exponentes matemáticos.

Ejemplo:

Hola <sup>mundo</sup>

El texto de arriba sería representado así:

Hola mundo

<sub>: Subíndex

Se utiliza para poner texto en posición de subíndice, comunmente usado en formulas químicas.

Ejemplo:

Hola <sub>mundo</sub>

El texto de arriba sería representado así:

Hola mundo
