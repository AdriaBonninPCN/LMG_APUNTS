Tablas en HTML

Tablas básicas en HTML

Etiqueta <table>:

  • La etiqueta <table> define la tabla i actua como el contenedor principal.

Etiqueta <tr> (table row):

  • Define una fila dentro de la tabla.

Etiqueta <td> (table data):

  • Define una celda dentro de una fila dentro de la tabla (datos de la tabla).

Ejemplo:

<table>
 <tr>
  <td>María</td>
  <td>25</td>
 </tr>
 <tr>
  <td>Juan</td>
  <td>30</td>
 </tr>
</table>

La tabla de arriba sería representada así:

María 25
Juan 30

Atributos de una tabla:

  • border:
    • Añade un borde a la tabla i a las celdas
  • cellpadding:
    • Define el espacio entre el contenido de la celda i el borde de la celda.
  • cellspacing:
    • Controla el espacio entre las celdas de la tabla

Ejemplo:

<table border="1" cellpadding="10" cellspacing="5">
 <tr>
  <td>Nombre</td>
  <td>Edad</td>
 </tr>
 <tr>
  <td>Juan</td>
  <td>28</td>
 </tr>
 <tr>
  <td>María</td>
  <td>32</td>
 </tr>
</table>

La tabla de arriba sería representada así:

Nombre Edad
Juan 28
María 32
