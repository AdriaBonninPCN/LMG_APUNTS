Diseño Básico con FlexBox

Flexbox es un sistema de diseño en CSS que proporciona una manera muy eficiente de distribuir el espacio entre los elementos y alinearlos dentro de un contenedor, incluso cuando el tamaño de los elementos es desconocido o dinámico. Flexbox es ideal para crear diseños de páginas adaptativas, ya que facilita la alineación, distribución y ordenación de los elementos dentro de un contenedor.

  1. El contenedor flex (display: flex): El punto de partida para utilizar Flexbox es establecer display: flex en el elemento contenedor. Este contenedor se conoce como contenedor flex, y sus elementos hijos se conocen como elementos flex.

    2. .contingut {
     display: flex;
    }

  2. Dirección del flujo (flex-direction): Por defecto, los elementos flex se disponen horizontalmente. Puedes cambiar la dirección con la propiedad flex-direction.
    • row: Los elementos se colocan en la dirección horizontal (valor por defecto).
    • row-reverse: Los elementos se colocan en la dirección horizontal, pero en orden inverso.
    • column: Los elementos se colocan en columna (dirección vertical).
    • column-reverse: Los elementos se colocan en columna, pero en orden inverso.

    3. .contenidor {
     display: flex;
     flex-direction: column;
    }

  3. Alineación horizontal (justify-content): La propiedad justify-content controla la alineación de los elementos a lo largo del eje principal (horizontal por defecto).
    • flex-start: Los elementos se alinean al principio del contenedor.
    • flex-end: Los elementos se alinean al final del contenedor.
    • center: Los elementos se centran dentro del contenedor.
    • space-between: Los elementos se distribuyen uniformemente con espacio entre ellos.
    • space-around: Los elementos tienen espacio alrededor de cada uno.

    4. .contenidor {
     display: flex;
     justify-content: center;
    }

  4. Alineación vertical (align-items): La propiedad align-items controla la alineación de los elementos a lo largo del eje transversal (vertical si la dirección es horizontal).
    • stretch: Los elementos se estiran para llenar el contenedor (valor por defecto).
    • flex-start: Los elementos se alinean en la parte superior del contenedor.
    • flex-end: Los elementos se alinean en la parte inferior del contenedor.
    • center: Los elementos se centran verticalmente dentro del contenedor.
    • baseline: Los elementos se alinean según su línea de base de texto.

    5. .contenidor {
     display: flex;
     align-items: center;
    }

  5. Control del espacio que ocupa cada elemento (flex-grow): La propiedad flex-grow especifica cómo los elementos flex crecen dentro del contenedor para ocupar el espacio disponible. Cada elemento puede asignarse un valor numérico que indica cuánto debe crecer en relación con los otros elementos.
    • Si todos los elementos tienen flex-grow: 1, el espacio disponible se distribuirá de manera uniforme entre ellos.

    6. .element {
     flex-grow: 2; /* Este elemento crecera el doble que los otros elementos*/
    }

Resum de les propietats vistes:

  • display: flex: Activa Flexbox en el contenedor.
  • flex-direction: Define si los elementos se colocan en fila (horizontal) o en columna.
  • justify-content: Alinea los elementos horizontalmente (eje principal).
  • align-items: Alinea los elementos verticalmente (eje transversal).
  • flex-grow: Controla cómo crece cada elemento dentro del contenedor para ocupar el espacio disponible.
