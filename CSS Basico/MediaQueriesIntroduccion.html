Media Queries (Introducción)

Las Media Queries son una característica poderosa de CSS que permite adaptar el diseño de una página web según las características del dispositivo que la está visualizando, como el tamaño de la pantalla, la resolución o la orientación. Esta técnica es clave para implementar diseños responsivos, que ofrecen una experiencia óptima tanto en dispositivos móviles como en tabletas o computadoras de escritorio.

  1. ¿Qué son las Media Queries? Una media query es una condición que se añade al CSS para aplicar reglas de estilo solo cuando se cumplan ciertas características del dispositivo o de la ventana del navegador. El ejemplo más común es aplicar estilos diferentes en función del ancho de la pantalla.

    2. @media (condició) {
     /* Estilos que solo se aplicaran si se cumple la condición */
    }

  2. Aplicación básica de Media Queries para pantallas de diferente tamaño Un uso típico de las media queries es adaptar el diseño a diferentes anchos de pantalla. Por ejemplo, puedes ajustar los estilos para pantallas grandes y pequeñas.

    3. /* Estilos para pantallas grandes (sobre los 768px de ancho) */
    @media (min-width: 768px) {
     body {
      background-color: lightblue;
     }
    }
    /* Estilos para pantallas pequeñas (bajo los 768px de ancho) */
    @media (max-width: 767px) {
     body {
      background-color: lightgreen;
     }
    }

  3. Tipos de condiciones más comunes en las Media Queries A continuación, veremos algunas de las condiciones más comunes que puedes utilizar dentro de las media queries:
    1. Ancho de la pantalla (min-width y max-width): Estas condiciones se utilizan para aplicar estilos cuando la ventana del navegador tiene un ancho mínimo (min-width) o máximo (max-width).
    2. Altura de la pantalla (min-height y max-height): Estas condiciones funcionan de manera similar a las de ancho, pero se basan en la altura de la ventana (min-height y max-height).
    3. Orientación de la pantalla (orientation): Puedes utilizar la condición orientation para aplicar estilos diferentes si la pantalla está en orientación horizontal (landscape) o vertical (portrait).


    @media (min-width: 600px) {
     /* Estilos para pantallas con un ancho de 600 píxeles o más */
    }
    @media (max-width: 600px) {
     /* Estilos para pantallas con un ancho de hasta a 600 píxeles */
    }
    @media (min-height: 500px) {
     /* Estilos para pantallas con un alto de 500 píxeles o más */
    }
    @media (orientation: landscape) {
     /* Estils pera dispositivos en orientación horizontal */
    }
    @media (orientation: portrait) {
     /* Estils pera dispositivos en orientación vertical */
    }

  4. Uso de múltiples condiciones en una Media Query Puedes combinar varias condiciones en una sola media query mediante las palabras clave and u only para especificar estilos muy concretos.

    5. @media (min-width: 600px) and (max-width: 1200px) {
     /* Estilos sólo aplicados si la pantalla tiene entre 600 y 1200 píxeles de ancho */
    }

Resumen de las propiedades vistas:

  • @media: Define una media query.
  • min-width y max-width: Aplica estilos según el ancho mínimo o máximo de la pantalla.
  • min-height y max-height: Aplica estilos según la altura mínima o máxima de la pantalla.
  • orientation: Aplica estilos según la orientación del dispositivo (vertical u horizontal).
  • and: Combina múltiples condiciones en una media query.
