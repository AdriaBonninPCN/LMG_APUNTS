Propiedades de Texto y Fuentes

El CSS permite controlar cómo se muestran los textos dentro de una página web. Con diversas propiedades, podemos cambiar la fuente, el color, la alineación, el tamaño del texto y otros aspectos visuales para mejorar la legibilidad y la estética del contenido.

  • Propiedad font-family: Define el tipo de letra (tipografía) que se utilizará para un elemento de texto. Puedes especificar una o más fuentes, y el navegador aplicará la primera que esté disponible en el sistema del usuario.

    • p {
     font-family: "Arial", sans-serif;
    }

  • Propiedad font-size: Controla el tamaño de la letra. Los valores se pueden definir en diversas unidades, como píxeles (px), ems (em), puntos (pt) o porcentajes (%).

    • h1 {
     font-size: 32px;
    }
    p {
     font-size: 14px;
    }

  • Propiedad color: Define el color del texto. Se pueden utilizar nombres de colores predefinidos, valores en hexadecimal o el sistema de colores RGB o RGBA.

    • p {
     color: red; /* Con nombre de color */
    }
    h1 {
     color: #00FF00; /* Con nombre hexadecimal */
    }
    span {
     color: rgb(0, 0, 255); /* Con codigo RGB */
    }

  • Propiedad text-align: Controla la alineación del texto dentro de un elemento. Los valores más comunes son:
    • left (izquierda)
    • right (derecha)
    • center (centrado)
    • justify (justificado)

    • h1 {
     text-align: center;
    }
    p {
     text-align: justify;
    }

  • Propiedad font-weight: Controla el grosor de la letra. Los valores comunes son:
    • normal: por defecto.
    • bold: texto en negrita.
    • Números entre 100 y 900, donde 400 es el valor por defecto y 700 representa negrita.

    • p {
     font-weight: bold;
    }

  • Propiedad font-style: Controla el estilo de la fuente, como si el texto será normal o en cursiva. Los valores más comunes son:
    • normal: texto normal.
    • italic: texto en cursiva.
    • oblique: texto inclinado.

    • span {
     font-style: italic;
    }
