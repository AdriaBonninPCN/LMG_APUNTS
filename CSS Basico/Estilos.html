El CSS se puede aplicar a un documento HTML de tres maneras principales: en línea, incrustado (dentro del HTML) y externo (mediante un archivo CSS separado). Cada una de estas formas tiene sus propósitos y usos.

  1. Estilos en línea (Inline styles) Los estilos en línea se aplican directamente a un elemento HTML, utilizando el atributo style dentro de la misma etiqueta del elemento. Este método es adecuado para aplicar estilos a un solo elemento de manera rápida, pero no es recomendable para páginas grandes, ya que dificulta el mantenimiento.

    2. <p style="color: red; font-size: 20px;">Este es un párrafo con estilos en línea.</p>

  2. Estilos incrustados (Embedded styles) Los estilos incrustados se colocan dentro de la sección <head> de un documento HTML, dentro de una etiqueta <style>. Estos estilos se aplican a todos los elementos HTML de la página actual.

    3. p {
     color: blue;
     font-size: 18px;
    }
    h1 {
     text-align: center;
    }

  3. Estilos externos (External stylesheets) Los estilos externos se almacenan en un archivo CSS separado, y se enlazan al documento HTML utilizando la etiqueta <link> dentro de la sección <head>. Este es el método más recomendado para la mayoría de sitios web, especialmente si constan de múltiples páginas.
