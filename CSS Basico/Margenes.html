Margenes, Rellenos y Bordes

Los márgenes (margin), los rellenos (padding) y los bordes (border) son parte del de el modelo de caja en CSS, que describe cómo se disponen los elementos en la página. Estas propiedades permiten ajustar el espacio exterior e interior de un elemento, así como definir sus límites visuales con una frontera.

  • Márgenes (margin): La propiedad margin controla el espacio exterior de un elemento, es decir, el espacio que hay entre un elemento y los otros elementos que lo rodean. Los márgenes se pueden definir de manera individual para cada uno de los cuatro lados del elemento (superior, derecho, inferior, izquierdo) o de manera global para todos los lados a la vez.

    • p {
     margin: 10px; /* Aplica un marge de 10 píxels a tots els costats */
    }
    h1 {
     margin-top: 20px; /* Margen superior de 20 píxeles */
     margin-bottom: 10px; /* Marge inferior de 10 píxeles */
     margin-left: 10px; /* Margen izquierdo de 10 píxeles */
     margin-right: 10px; /* Margen derecho de 10 píxeles */
    }

  • Rellenos (padding): La propiedad padding controla el espacio interior de un elemento, es decir, el espacio entre el contenido de un elemento (como texto o imágenes) y sus fronteras. Al igual que con los márgenes, los rellenos se pueden definir para todos los lados o para cada uno individualmente.

    • div {
     padding: 20px; /* Aplica un relleno de 20 píxeles a todos los lados */
    }
    h2 {
     padding-left: 15px; /* Aplica un relleno de 15 píxeles al lado izquierdo */
    }

  • Bordes (border): La propiedad border se utiliza para añadir un borde alrededor de un elemento. Esta propiedad tiene varios subatributos:
    • Ancho del borde (border-width): define el grosor del borde.
    • Estilo del borde (border-style): puede ser solid (sólido), dashed (rayado), dotted (puntos), etc.
    • Color del borde (border-color): especifica el color del borde.

    • div {
     border: 2px solid black; /* Borde de 2 píxeles, sólido i negro */
    }
    p {
     border: 1px dashed blue; /* Borde d'1 píxel, a rayas i azul */
    }

Unidades comunes

  • Píxeles (px): Unidad fija, representa un punto en la pantalla. Ejemplo: margin: 20px;
  • Ems (em): Unidad relativa al tamaño de la fuente del elemento padre. Ejemplo: padding: 2em; (2 veces el tamaño de la fuente).
  • Rems (rem): Unidad relativa al tamaño de la fuente del elemento raíz (html). Ejemplo: margin: 1.5rem; (1.5 veces el tamaño de la fuente raíz).
  • Porcentaje (%): Relativo al ancho o alto del contenedor padre. Ejemplo: padding: 5%; (5% del ancho del contenedor).
  • Viewports (vw, vh): Relativo al tamaño de la ventana visible. Ejemplo: margin: 5vw; (5% del ancho del viewport).
  • Bordes (border-width): Grosores predeterminados: thin, medium, thick o valores como px. Ejemplo: border: thick solid black;
