Sintáxis Básica CSS

  • Regla CSS: és una instrucció que defineix com s'ha d'aplicar un estil a un o més elements HTML dins d'una pàgina web.
  • Selector: Indica l'element HTML al qual s'aplicaran els estils.
  • Propietat: Defineix la característica que vols modificar (com ara el color, la mida del text, el marge, etc.).
  • Valor: Assigna un valor específic a la propietat (com ara "blau" per al color, "16px" per a la mida del text, etc.).
  • Declaració CSS: és una instrucció dins d'una regla CSS que defineix una característica específica d'estil per a un element HTML.

Selectores en CSS

Los selectores son una parte fundamental del CSS porque definen qué elementos HTML serán afectados por las reglas de estilo. Hay varios tipos de selectores, pero los más comunes en el aprendizaje básico son:

  • Selector de tipo (Type Selector): Este selector aplica estilos a todos los elementos HTML del mismo tipo. Este código aplicará el color verde a todos los títulos (<h1>) de la página.

    • Ejemplo:

    h1 {
      color: green;
    }

  • Selector de clase (Class Selector): Los selectores de clase permiten aplicar estilos a elementos HTML con una clase específica. Se utiliza un punto (.) para seleccionar una clase. Cualquier elemento HTML que tenga el atributo class="mi-classe" tendrá un fondo amarillo.

    • Ejemplo:

    .mi-classe {
     background-color: yellow;
    }

    <p class="mi-classe">Este párrafo tendra el fondo amarillo</p>

    Este párrafo tendra el fondo amarillo

  • Selector de id (ID Selector): El selector de id se utiliza para aplicar estilos a un único elemento HTML que tenga un identificador específico. Se usa el símbolo de la almohadilla (#). Este ejemplo centrará el texto del elemento que tenga id="mi-id". Es importante recordar que un ID debe ser único en la página.

    • Ejemplo:

    #mi-id {
     text-align: center;
    }

  • Selector universal: El selector universal (*) aplica estilos a todos los elementos de una página. Este ejemplo elimina los márgenes y rellenos de todos los elementos de la página, lo cual es útil para establecer un estilo uniforme.

    • Ejemplo:

    * {
    margin: 0;
    padding: 0;
    }

Conceptos importantes

  • Herencia: Los elementos hijos heredan los estilos de sus padres, por lo que no es necesario escribir estilos para todo.
  • Especificidad: Cuando hay conflictos de estilos, el navegador aplica solo el de mayor especificidad.
  • Cascada: Todo estilo sobrescribe uno anterior. La C de CSS significa Cascada.
