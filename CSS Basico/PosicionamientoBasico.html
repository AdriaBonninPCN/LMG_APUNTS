Posicionamiento Básico

CSS ofrece diversas técnicas para controlar la ubicación de los elementos en una página web. La propiedad clave para el posicionamiento es position, que tiene varios valores para definir cómo se posiciona un elemento respecto a su contenedor o al documento.

  1. La propiedad position: Permite especificar el método de posicionamiento de un elemento. Los valores más comunes son:
  2. Propiedades de posicionamiento (top, right, bottom, left): Cuando se define la propiedad position como relative, absolute, fixed o sticky, se pueden usar las propiedades top, right, bottom y left para especificar la distancia del elemento respecto a sus límites.
    • top: 10px; Mueve el elemento 10 píxeles hacia abajo desde el límite superior.
    • right: 20px; Mueve el elemento 20 píxeles hacia la izquierda desde el límite derecho.

    3. div {
     position: relative;
     top: 20px; /* Mueve el elemento 20 píxeles hacia abajo desde su posición inicial */
     left: 30px; /* Mueve el elemento 30 píxeles hacia la izquierda */
    }

  3. Posicionamiento absoluto (position: absolute): El elemento se saca completamente del flujo normal del documento y se posiciona respecto a su elemento contenedor más cercano con position: relative, absolute o fixed.

    4. div {
     position: absolute;
     top: 50px;
     left: 100px;
    }

  4. Posicionamiento fijo (position: fixed): Los elementos con position: fixed se mantienen en una ubicación fija en la pantalla incluso cuando el usuario hace desplazamientos. Se posicionan respecto a la ventana del navegador.

    5. div {
     position: fixed;
     top: 0;
     left: 0;
     width: 100px;
     height: 100px;
     background-color: red;
    }

  5. Posicionamiento pegajoso (position: sticky): Los elementos con position: sticky se comportan como relative mientras el usuario se desplaza por la página y llega a un punto especificado, después del cual el elemento se comporta como fixed.

    6. h1 {
     position: sticky;
     top: 10px;
    }

  6. Propiedad z-index: Se usa para controlar el orden de superposición de los elementos posicionados (relative, absolute, fixed o sticky). Los valores más altos de z-index colocan el elemento delante de otros elementos con valores inferiores.

    7. div {
     position: absolute;
     z-index: 10; /* Aquest element es mostrarà davant dels altres amb z-index inferior */
    }

Resumen de propiedades vistas:

  • position: static: Posicionamiento por defecto.
  • position: relative: El elemento se mueve respecto a su posición original.
  • position: absolute: El elemento se posiciona respecto a su contenedor más cercano.
  • position: fixed: El elemento se fija respecto a la ventana del navegador.
  • position: sticky: El elemento se comporta como relativo hasta que llega a una posición especificada, donde se mantiene fijo.
  • z-index: Controla el orden de superposición de los elementos.
