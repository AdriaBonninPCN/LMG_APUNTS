Dimensiones y Cajas

El modelo de caja: El CSS utiliza el modelo de caja para representar los elementos de una página web. Cada elemento se considera como una caja rectangular que contiene el contenido, rellenos, márgenes y bordes. Es importante entender este modelo porque controla cómo se distribuyen y ajustan los elementos dentro de la página. Partes principales del modelo de caja:

  1. Contenido: El contenido real del elemento, como texto, imágenes u otros elementos.
  2. Relleno (padding): El espacio entre el contenido y el borde.
  3. Borde (border): El límite alrededor del relleno.
  4. Margen (margin): El espacio externo entre el borde y otros elementos.
  1. Propiedades de dimensiones (width y height): Las propiedades width (ancho) y height (alto) se utilizan para definir el tamaño de un elemento. Estos valores se pueden definir en unidades fijas como píxeles (px) o unidades relativas como porcentajes (%), que dependen del tamaño de su elemento contenedor.

    2. div {
     width: 300px; /* Altura fija de 300 píxeles */
     height: 150px; /* Altura fija de 150 píxeles */
    }
    .container {
     width: 50%; /* Altura del 50% de el elemento contenedor */
     height: auto; /* La altura se ajusta autmáticamente*/
    }

  2. El modelo de caja en CSS: Establece cómo se calculan las dimensiones de un elemento teniendo en cuenta las propiedades de márgenes, rellenos y bordes. Por defecto, el tamaño de width y height solo afecta al contenido. Para incluir el relleno y el borde dentro de las dimensiones del elemento, se puede utilizar la propiedad box-sizing.
    3. Propiedad box-sizing: Controla cómo se calculan las dimensiones totales del elemento. Hay dos opciones principales:
    • content-box (valor por defecto): Solo incluye el contenido en las dimensiones, mientras que los bordes y los rellenos se añaden aparte.
    • border-box: Incluye el contenido, el relleno y el borde dentro de las dimensiones especificadas.

    div {
     width: 200px;
     padding: 20px;
     border: 5px solid black;
     box-sizing: border-box;
    }

  3. Control del desbordamiento (overflow): Cuando el contenido de un elemento es más grande que las dimensiones especificadas, el CSS ofrece la propiedad overflow para controlar cómo se muestra el contenido que desborda. Los valores más comunes son:
    • visible: El contenido que desborda se muestra fuera del elemento (valor por defecto).
    • hidden: El contenido que desborda se recorta y no se muestra.
    • scroll: Muestra barras de desplazamiento, aunque el contenido no desborde.
    • auto: Muestra barras de desplazamiento solo si el contenido desborda.

    4. div {
     width: 150px;
     height: 100px;
     overflow: scroll;
    }

Resumen de las propiedades vistas:

  • width y height: Definen las dimensiones de un elemento.
  • box-sizing: Controla cómo se calculan las dimensiones totales de un elemento, teniendo en cuenta el relleno y el borde.
  • overflow: Controla cómo se maneja el contenido que desborda las dimensiones del elemento.
